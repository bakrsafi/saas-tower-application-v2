# Interactive Real Estate Platform

An interactive real estate platform designed for property developers to create and present real estate projects using real building images with an interactive Canvas layer. End users can explore buildings visually, click on individual units, and view detailed apartment information.

The platform consists of four main modules.

---

# 1. Builder

The internal system used by administrators or project managers to create and manage real estate projects.

## 1.1 Data Entry

### Project Management

- Create Project
- Edit Project
- Delete Project

---

### Building Management

- Create Building
- Edit Building
- Delete Building

---

### Apartment Management

- Create Apartment
- Edit Apartment
- Delete Apartment

---

## 1.2 Building Workspace

The visual workspace used to prepare the interactive building experience.

### Building Image Management

- Add Building Image
- Edit Building Image
- Delete Building Image

---

### Polygon Management

- Draw Polygon
- Edit Polygon
- Delete Polygon

---

### Apartment ↔ Polygon Mapping

- Link apartments to polygons using **Drag & Drop**.

---

### Apartment List

- Display all apartments belonging to the selected building.
- Drag an apartment onto its corresponding polygon.
- Highlight apartments that have already been mapped.

---

### Workspace

- Display the building image.
- Display the Canvas layer.
- Zoom In / Zoom Out.
- Navigate between building images (Next / Previous).

---
